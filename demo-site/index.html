Physiological Flow: Pace bonuses incentivize moderate-vigorous intensity (65-85% HR max).

Self-Determination Theory

Autonomy: No mandatory schedules - run when and where you want.

Competence: Adaptive challenges and clear progress visualization.

Relatedness: Asynchronous competition and alliance systems.

GameFlow Model

Minimalist UI for concentration during exercise, real-time feedback, scalable challenges, and clear goals.

Malone's Framework

Challenge: Dynamic defense formulas create uncertain outcomes.

Fantasy: Territorial conquest directly tied to physical performance.

Curiosity: Fog-of-war exploration and strategic depth.
